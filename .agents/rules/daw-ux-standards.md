# DAW UX Standard

This is not a marketing site.
This is a production tool.

UX must prioritize:

- Speed
- Precision
- Clarity
- Workflow efficiency

Rules:

- Avoid decorative animations
- All controls must feel immediate
- Sliders and knobs must be precise
- No delayed UI transitions during playback
- Transport must always remain visible
- Timeline is primary focus

Panels:

- Mixer collapsible on small screens
- Inspector collapsible
- Piano roll scrollable
- Timeline zoomable
