# Responsive DAW Layout Rule

Mobile-first approach.

Breakpoints:

Mobile:
- Vertical stacked layout
- Mixer in drawer
- Timeline scroll horizontal
- Compact transport

Tablet:
- Two-column layout
- Collapsible mixer

Desktop:
- Full DAW layout
- Timeline + Tracks + Mixer visible

Never break grid alignment.
Never overflow UI.
Touch interactions must work.
